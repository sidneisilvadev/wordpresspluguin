Plugin avançado para WordPress que utiliza Inteligência Artificial para gerar conteúdo de alta qualidade, com suporte a múltiplos modelos de IA.

== Description ==

O AI Content Generator é um plugin poderoso que utiliza modelos avançados de IA da Groq para ajudar você a criar conteúdo de alta qualidade para seu site WordPress.

= Principais Recursos =

* Geração de conteúdo com IA avançada
* Suporte a múltiplos modelos de IA (Mixtral, LLaMA2, Gemma)
* Otimização de SEO integrada
* Geração de palavras-chave relacionadas
* Criação de esboços de artigos
* Geração de FAQs
* Sistema de cache para melhor performance
* Interface moderna e intuitiva
* Suporte a tema escuro
* Totalmente responsivo

= Como Usar =

1. Instale e ative o plugin
2. Configure sua chave da API do Groq em Configurações > AI Content Generator
3. Acesse o editor de posts/páginas
4. Use o botão "AI Content Generator" para começar a gerar conteúdo

= Requisitos =

* WordPress 5.6 ou superior
* PHP 7.4 ou superior
* Chave da API do Groq (obtenha em groq.com)

== Installation ==

1. Faça upload dos arquivos do plugin para a pasta `/wp-content/plugins/`
2. Ative o plugin através do menu 'Plugins' no WordPress
3. Configure sua chave da API em Configurações > AI Content Generator

== Frequently Asked Questions ==

= Como obtenho uma chave da API do Groq? =

Visite groq.com e crie uma conta para obter sua chave da API.

= Quais modelos de IA estão disponíveis? =

* Mixtral 8x7B (Recomendado)
* LLaMA2 70B
* Gemma 7B

= O plugin funciona com o editor Gutenberg? =

Sim, o plugin é totalmente compatível com o editor Gutenberg.

= Como funciona o sistema de cache? =

O plugin armazena as respostas da IA em cache para melhorar a performance e economizar tokens. O cache pode ser habilitado/desabilitado nas configurações.
